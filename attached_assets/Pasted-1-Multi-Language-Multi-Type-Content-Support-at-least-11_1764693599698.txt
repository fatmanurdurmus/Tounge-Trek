1. Multi-Language, Multi-Type Content
•	Support at least 11 languages with an architecture that can expand easily.
•	Content types:
proverb, idiom, legend, myth, folkStory, epic, mythologyNote, regionalTale, meme, travelPhrase, dailyExpression, trending_quote.
•	All entries follow the Content interface above and include rich metadata: translation, meaning, cultural context, tags, region, popularity, etc.
•	Implement semantic relationship mapping using relatedContent and shared tags to connect:
o	Myths ⇄ proverbs
o	Memes ⇄ daily expressions
o	Epics ⇄ regional tales
•	Support mythology explorer style browsing: by culture, era, themes, characters.

Tabs: Home, Explore, Travel, Daily, Library, Profile

o	Daily featured expression card (“Expression/Quote of the Day”)

o	Streak counter & quick progress indicators
o	Search bar with auto-suggestions
o	Theme toggle (Light / Dark / System)
•	Explore Screen
o	Header search bar with live results using FTS5

	Filters: language(s), content type, category, region, tags, popularity, date added
	Sort: alphabetical, popularity, date added, recently viewed
o	Grid/List toggle for content cards
o	Semantic “Related expressions” section on detail views
•	Travel Mode Screen
o	Country / language selector

o	Curated TravelPhrase content: survival phrases, etiquette tips, common memes/phrases one might hear
o	Offline access for downloaded travel sets
o	One-tap copy and optional TTS playback (expo-speech)
•	Daily Screen
o	Queue of “Today’s 5” expressions (mix of proverbs/idioms/memes)
o	Options:
	Mark as learned
	Quick quiz mode
o	Streak display & XP gains
o	Button to set daily notification time
•	Library Screen
o	Sections: Bookmarks, Downloads (offline), Recently Viewed
o	Show offline badges for cached items
o	Filters and sort options
•	Profile / Settings Screen

	Learned count per category/language
	Time-on-task
	Quiz accuracy

o	Settings:
	Theme (Light/Dark/System)

	Upload Meme (image picker → Supabase upload → create local entry)
	Open Achievements and Leaderboards
	Open Feedback / “Report an issue”

o	Show full text, translation, meaning, cultural context, tags, type, language

o	Semantic relationships: “Related expressions” & “Related myths/stories”
o	Buttons: Favorite, Bookmark, Share, Report/Feedback

•	Additional Screens
o	AchievementsScreen: grid of badges with unlock status
